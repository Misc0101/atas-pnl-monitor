# ATAS PnL Monitor — 使用手册

## 运行环境要求

- Windows 10 / 11（x64）
- [ATAS](https://atas.net/) 8.0 或更高版本，已安装且至少有一笔成交记录

---

## 安装与启动

### 方式 A：便携版（推荐）

1. 前往 [Releases](../../../releases) 下载 `ATAS-PnL-Monitor-portable.exe`
2. 放到任意位置，无需安装
3. 双击直接运行

### 方式 B：安装包

1. 从 [Releases](../../../releases) 下载 `ATAS PnL Monitor Setup x.x.x.exe`
2. 运行安装包，会自动创建桌面快捷方式
3. 通过桌面快捷方式或开始菜单启动

---

## 首次启动

程序启动时会自动：

- 定位 ATAS 成交记录文件（`%APPDATA%\ATAS\Database\HistoryMyTrade.cdb`）
- 将开始时间设为当前时刻，并启用**至今**模式
- 勾选过去 24 小时内有成交记录的所有账户

若没有数据显示，请检查侧边栏的**数据文件**区块（见下文）。

## 侧边栏 — 筛选区

### 活跃账户

列出过去 24 小时内有平仓记录的账户。

| 按钮 | 功能 |
|------|------|
| **全选** | 勾选所有活跃账户 |
| **清空** | 取消所有账户的勾选 |
| **所有账户（N）** | 打开全量账户弹窗 |

**全量账户弹窗** — 展示所有账户（活跃 + 历史），分组显示。每行包含：
- 勾选框，控制该账户是否纳入图表统计
- **全程**按钮 — 将时间范围设为该账户第一笔至最后一笔成交
- **▶** 按钮 — 展开该账户的交易时段列表，每段附**查看**按钮可一键跳转

> **时段间隔阈值**（弹窗内齿轮图标）：程序通过相邻成交间的空档时长自动划分交易时段，默认 8 小时。根据你实际的交易时间段进行调整。

### 品种

选择**全部**查看所有品种的合并曲线，或选择单一品种单独分析。

### 时间范围

| 控件 | 说明 |
|------|------|
| **开始**时间 | 显示窗口的起点 |
| **4h / 8h / 24h** | 将开始时间设为当前往前推对应小时，结束切为至今 |
| **至今** | 结束时间实时跟随当前时刻，有新成交时图表自动延伸 |
| **结束**时间 | 手动设置截止时间，自动退出至今模式 |

### 时区偏移

ATAS 数据文件中所有时间均为 UTC+0。将时区偏移设为你所在地区的 UTC 偏移，图表 X 轴会还原为本地时间，便于复盘。

*示例：UTC+8 — 文件中记录的 UTC 12:00 在图表上显示为 20:00。*

### 数据文件

显示当前使用的数据文件路径。若文件无法读取，以红色显示错误提示。

| 按钮 | 功能 |
|------|------|
| **更改** | 手动选择其他 `.cdb` 文件 |
| **恢复默认** | 恢复至 ATAS 默认路径（仅在使用自定义路径时显示） |

## 图表

### 双曲线

| 曲线 | 颜色 | 含义 |
|------|------|------|
| 盈亏曲线（不含手续费） | 珊瑚红 | `PnL` 字段的累计值 |
| 净盈亏（含手续费） | 钢蓝 | `PnL + Commission` 的累计值 |

鼠标悬停于图表任意位置，浮窗显示该时间点两条曲线的精确值。

### 最大回撤

若存在回撤，图表会以半透明红色区域标注峰值到谷值的时间段，并在左上角标注回撤金额。

### 净盈亏数值叠加（仅看曲线模式）

在仅看曲线模式下，图表顶部始终显示当前**净盈亏**数值（绿色 = 盈利，红色 = 亏损）。当多笔单子集中平仓、曲线末端被压缩成近似竖线时，可直接读取该数值，无需从 Y 轴估算。

---

## 仅看曲线模式

点击标题栏的**仅看曲线**按钮，隐藏侧边栏、概览行和品种统计，仅保留图表。窗口自动缩小为紧凑尺寸。

- 该模式的窗口大小和位置与完整模式**独立保存**
- 所有筛选条件保持不变，随时切回完整模式无需重新设置
- 配合**置顶**使用，可将图表悬浮在交易软件上方

---

## 置顶

点击标题栏的**置顶**按钮，窗口始终浮于其他窗口之上。可与仅看曲线模式独立组合使用。

## 品种统计表

图表下方按品种分组显示统计数据：

| 列 | 说明 |
|----|------|
| 品种 | 合约名称 |
| 交易次数 | 已平仓笔数 |
| 胜率 | 净盈亏 > 0 的笔数 ÷ 总笔数 |
| 盈亏比 | 平均盈利 ÷ 平均亏损（绝对值） |
| 平均盈利 | 盈利笔的平均净盈亏 |
| 平均亏损 | 亏损笔的平均净盈亏（显示为负值） |
| 毛盈亏 | 该品种所有 `PnL` 之和 |
| 净盈亏 | 该品种所有 `PnL + Commission` 之和 |

拖动表格上方的分隔条可调整面板高度。可用滚动条或鼠标滚轮横向滚动表格。

## 常见问题

**没有数据 / 显示"当前筛选条件下暂无数据"**
- 确认侧边栏至少勾选了一个账户
- 检查时间范围是否覆盖了有成交的时段
- 确认数据文件路径正确（侧边栏 → 数据文件区块）

**提示文件不存在或无法读取**
- 确保 ATAS 已安装且至少产生过一笔成交
- 若你移动过 ATAS 数据目录，点击**更改**重新指定路径

**X 轴时间显示不对**
- 调整侧边栏的**时区偏移**，设为你所在地区的 UTC 偏移值
